# Commit.exe 

**Commit.exe** is a fun application that generates humorous Git commit messages whenever you're out of ideas—or just want to make your repository a little more entertaining.

## Features

-  Generates random funny Git commit messages
- One-click message generation
- Generate unlimited commit messages

## Example Messages

- `Fixed the bug... hopefully.`
- `It works on my machine.`
- `Added more duct tape.`
- `This should never have worked, but it did.`
- `Removed code that was offending the compiler.`
- `Magic happened here `
